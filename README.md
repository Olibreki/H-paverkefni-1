# Hópaverkefni 1

## Keyrsla verkefnis.
Keyrir Sass og fylgist með breytingum:
npm run dev
eða 
npm start

Framleiðslu build fæst með npm run build

### Lint
npm run lint

Til að laga villur sjálfkrafa:
npm run lint:scss:fix
